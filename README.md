# Project Grass

It`s 2D game constructor built with Vue and Electron

### Project setup
```
npm install
```

Before running/building app you must place golang compiler in mingo/ directory
### Run app
```
electron:serve
```
### Build app
```
electron:build
```
### Run your tests
```
test:unit
```

#### Run Vue app
```
vue-cli-service serve
```
